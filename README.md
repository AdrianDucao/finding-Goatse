# finding-PORTy
This is a proof of concept for educational purposes only port scanning tool.

### Features:
1. Scans all open port
2. Scans for live host
3. TCP Scan
4. UDP Scan

### Installation and Usage:
```
$ git clone https://github.com/AdrianDucao/finding-PORTy.git

$ cd finding-PORTy

$ ./scan.py

```
